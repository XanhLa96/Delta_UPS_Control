1. Run Your UPS Monitor Service (Flask_RestfulApi_UPS_Serial.py)
2. Find Your UPS Monitor Host IP & Port
3. Change "http_Response.py" python File Service IP & Port
4. Open the terminal Enter command "python check_ups_status.py"
5. Get the UPS SystemMode Status Return
6. Check the code can run in your Icinga2 host computer
7. Copy the file "check_ups_status.py" to your Icinga Plugin script Folder (/etc/icinga2/scripts/)
8. Copy the command.txt content add to Icinga Plugin command config File (/etc/icinga2/conf.d/commands.conf)
9. Copy the service.txt content add to Icinga Plugin service config File (/etc/icinga2/conf.d/services.conf)
10.Copy the host.txt content add to Icinga Plugin host config File (/etc/icinga2/conf.d/hosts.conf)
11.Open Terminal Enter command "systemctl restart icinga2"
12.Open Terminal Enter command "systemctl status icinga2 -l" to debug the Error !
